  • Welcome to StayHealthy

    Unsere Software StayHealthy soll Ihnen dabei helfen ihr Gesundheitsverhalten zu verbesser, in dem es Ihnen dabei hilft ihr Ernährungs- und Trainingsverhalten zu dokumentieren und eine Übersicht darüber gibt wieviele Kalorien Sie zu sich genommen bzw. verbraucht haben. Außerdem kann die Software individuell Trainingspleane vorschlagen.

    Auf dieser Website erhalten Sie einen Überblick über unser Produkt und dessen Entwicklung. Außerdem stellen wir eine kostenlose Version von StayHealthy zum Download zur Verfügung.

    Die Anwendung ist zurzeit nur innerhalb des OTH Netzwerkes verwendbar.

  • About the Project

    Das Projekt umfasst insgesammt 2393 LOC in der Programmiersprache C++. Diese sind auf 20 Klassen aufgeteilt. Für die GUI wurde das Qt Framework verwendet. Als Datenbank verwenden wir Microsoft SQL Server. Als IDE haben wir Microsoft Visual Studio 2022 Community Edition eingesetzt. Außerdem haben wir LateX zur Formatierung unserer Dokumente benutzt.

    Alle Dokumente und Quelldateien sind auffindbar unter https://git.manherz.de/denismhz/SWP
